# 安装与快速开始

先决条件：Node.js 18+，npm

1. 克隆仓库：

```powershell
git clone https://github.com/hverlott/slotgamerougelike.git
cd slotgamerougelike
```

2. 安装依赖并运行开发服务器：

```powershell
npm install
npm run dev
```

3. 生产构建：

```powershell
npm run build
```

常见问题：
- 如果 `npm ci` 报错，请先运行 `npm install` 同步 lockfile。
