# d-slotgame (Slot Game Rouge-like)

轻量的 HTML5 插槽游戏仓库。

主要改进：
- 版本管理：使用 `standard-version` 管理语义化版本与 `CHANGELOG.md`。
- 发布：使用 GitHub Releases（通过 tag `v*` 触发工作流）。
- 文档：添加基础贡献与文档目录。

常用命令：

```powershell
npm install
npm run dev
npm run build
npm run release   # 本地生成版本并更新 CHANGELOG
```

查看 `docs/` 目录获取更多文档。
